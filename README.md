## Proyectos

### [Regresión Lineal con Base de Datos](./Regresion_Lineal/)

Implementación de un modelo de **regresión lineal simple** conectado a MySQL, desarrollado en tres lenguajes y visualizado en un sistema web multi-página con navegación.

**Tecnologías:** PHP 8 · Python 3 · C · MySQL · Chart.js

**Características:**
- Dashboard con tarjetas de métricas (R², MSE, RMSE) y gráfica interactiva
- Gestión de datos: agregar y eliminar registros con detección de outliers (±2.5σ)
- Página de modelo: ecuación `Y = b0 + b1·X` y bondad de ajuste
- Calculadora de predicción
- Consultas seguras con sentencias preparadas (anti SQL Injection)
